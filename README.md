# SZTU新教评脚本安装指南

## 📌 操作步骤

1. **安装篡改猴扩展**  
👉 点击链接安装：[篡改猴 Edge 扩展](https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/iikmkjmpaadaobahmlepeloendndfphd)  
（在扩展页面点击"获取"按钮完成安装）

2. **开启开发人员模式**
👉 点击浏览器右上角···-扩展-管理扩展  
✅ 打开左侧 **开发人员模式** 开关

3. **打开篡改猴管理面板**  
🔧 左键点击右上角**浏览器扩展栏**中的篡改猴图标  
→ 选择"管理面板"

4. **安装教评脚本**  
🖱️ 在右侧release下载脚本（.js），直接拖拽到篡改猴**管理面板**中  
（弹出安装提示时点击**安装**按钮）

5. **访问教评系统**  
🌐 打开[教评链接，需要先登录](https://ddpj.sztu.edu.cn/index.html?v=3.35.0#/my-task/main/UnFinished)【[教务](https://auth.sztu.edu.cn/idp/authcenter/ActionAuthChain?entityId=home)】| [校外链接](https://home-sztu-edu-cn-s.webvpn.sztu.edu.cn:8118/bmportal/index.portal)  
💡 脚本会自动激活（观察篡改猴图标显示数字即表示运行中）

6. **开始教评**  
📝 按照脚本提示完成教评任务即可  
❗ Auto版本仅支持一键好评，且在单课程超过三位老师的情况下无法正常运行，请手动刷新。Optional版本支持一键好评和一键差评（慎重选择）。
